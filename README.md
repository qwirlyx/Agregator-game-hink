# Game Aggregator

Лендинг выбора площадки для запуска мини-игры. Показывает кнопки мессенджеров и подгружает ссылки из JSON-конфига.

## Что умеет

- показывает страницу выбора мессенджера;
- берёт ссылки из `data/messengers.json` или из пути, указанного в переменной окружения `GAME_CONFIG_PATH`;
- поддерживает Telegram, MAX, VK и OK;
- может работать как отдельная посадочная страница рядом с основным проектом игры.

## Стек

- HTML
- CSS
- JavaScript
- PHP для отдачи JSON-конфига

## Запуск

1. Загрузить файлы на хостинг.
2. Проверить ссылки в `data/messengers.json`.
3. При необходимости задать `window.GAME_BASE_URL` перед подключением `assets/js/app.js`, если игра лежит не в `/hinkjump/`.
4. Открыть `index.html`.

## Важные файлы

- `index.html` — главная страница;
- `assets/js/app.js` — загрузка настроек и подстановка ссылок;
- `data/messengers.json` — список площадок и ссылок;
- `config.php` — отдача JSON-конфига.
